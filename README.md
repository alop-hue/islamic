# Islamic Browser

متصفح سطح مكتب بسيط مبني باستخدام **Electron** (المعتمد على Chromium).

## المزايا
- شريط أدوات للتنقل (رجوع/تقدم/تحديث/الرئيسية)
- شريط عنوان مع دعم إدخال رابط أو كلمة بحث
- يعتمد على Chromium عبر Electron

## التشغيل
```bash
npm install
npm start
```
